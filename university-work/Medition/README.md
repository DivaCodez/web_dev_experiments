## 🧠 Key Concepts Learned

### 1. HTML Structure & Content
*   **HTML Basics:** Understanding the skeleton of a webpage (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
*   **Hyperlinks:**
    *   **External Links:** Linking to outside websites using absolute URLs.
    *   **Internal Links:** Creating navigation within the same page or relative paths to local files.
*   **Media Integration:**
    *   **Adding Photos:** Inserting images using the `<img>` tag and managing paths.
*   **Data Organization:**
    *   **Tables:** Structuring tabular data using `<table>`, `<tr>`, `<th>`, and `<td>`.

### 2. Code Organization (Containers)
*   **Block-level Containers:** Using `<div>` to group larger sections of content and create layouts.
*   **Inline Containers:** Using `<span>` to target specific pieces of text within a block for styling.
*   **Semantic Structure:** Learning how to organize code efficiently for readability and maintenance.

### 3. CSS Styling Fundamentals
*   **Internal CSS (`<style>`):** Writing CSS rules directly inside the HTML document within the `<style>` tag.
*   **Attributes & Selectors:**
    *   Targeting elements by tag name, class, and ID.
    *   Applying basic attributes like colors, fonts, borders, and backgrounds.

## 🛠️ Technical Summary

| Feature | Tag / Concept | Usage |
| :--- | :--- | :--- |
| **External Link** | `<a href="url">` | Redirect to another website. |
| **Internal Link** | `<a href="#id">` | Navigate within the same page. |
| **Images** | `<img src="path" alt="text">` | Display visual content. |
| **Data Tables** | `<table>` | Display structured data (rows/cols). |
| **Block Container** | `<div>` | Group elements for layout. |
| **Inline Container** | `<span>` | Style specific text parts. |
| **Internal Style** | `<style>` | Embed CSS in HTML head. |
